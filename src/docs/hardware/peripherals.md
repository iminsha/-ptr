# 硬件资源与外设依赖（以板载/预留为准）

## 必选外设（项目核心闭环）

- MCU：普中51（STC89C52/兼容）
- RTC：DS1302（时钟日期）
- 温度：DS18B20（单总线）
- EEPROM：AT24C02（配置与规则存储）
- 显示：LCD1602（状态/设置界面）
- 输入：独立按键/矩阵键盘（二选一或混用）
- 执行：ULN2003 + 五线四相步进电机接口
- 提示：蜂鸣器
- 通信：CH340 串口（日志/CLI 可选）

## 推荐外设（提升交互与演示效果）

- 红外接收头（NEC遥控控制 / 快捷场景）
- 数码管/LED/点阵（可做“开度条、动画、告警闪烁”）

## 约束与边界

- 项目默认不涉及大功率负载（无继电器/加热驱动设计）。
- 若后续扩展外部负载，必须明确隔离与安全策略（本文档不覆盖）。
