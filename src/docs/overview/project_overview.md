# 项目概述

本项目基于普中 51 开发板已有外设，实现一个可独立运行的“控制终端”。

核心能力：

- 实时显示：时间 / 温度 / 模式 / 执行器位置
- 可编程自动化：按“时间日程 + 温度规则”触发动作
- 本地人机交互：LCD1602 + 按键/矩阵键盘 + 红外遥控
- 执行器控制：步进电机（ULN2003 驱动）可定位到百分比开度
- 配置持久化：24C02 保存规则、阈值、映射，掉电恢复
- 日志输出：串口输出事件、错误、状态快照，便于验收与调试
- 告警提示：蜂鸣器提示超温、故障、任务触发

目标不是“把模块都点亮”，而是完成一个结构清晰、可维护、可扩展的小型嵌入式系统。
