# 功能规格（用户视角）

## 模式

- AUTO（自动）：按规则/日程自动执行动作
- MANUAL（手动）：按键/遥控直接控制执行器（开/关/到某位置）
- SAFE（安全）：出现关键错误（温度传感器失效、EEPROM校验失败等）进入安全模式，仅允许停止与恢复

## 规则系统

规则由两类触发组成：

时间日程任务（Schedule）

- 每日定时：HH:MM 触发
- 周期任务：间隔 N 分钟触发（可选）
- 任务动作：MoveTo/MoveStep/Beep/LEDPattern/ModeSwitch…

温度规则（Temp Rule）

- 高阈值触发（例如 T > TH）
- 低阈值恢复（例如 T < TL）
- 必须支持：滞回（TH/TL）+ 冷却时间（cooldown）避免抖动

## 执行器能力

位置表达：0%~100%（映射到步数）

指令：

- MOVE_TO(pct)：移动到目标开度
- STOP()：立即停止
- HOME()：回零（若无硬限位：软件回零 + 用户确认）
- SET_SPEED(level)：速度档位（低/中/高）

## 日志与告警

- 事件日志：任务触发、规则触发、手动操作、错误发生/恢复
- 错误码体系：可从串口导出用于验收复现
- 蜂鸣器：对关键告警与用户确认提示
